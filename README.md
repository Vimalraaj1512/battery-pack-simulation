# Battery Pack Modeling and Simulation using Simulink

This project focuses on modeling and simulating a lithium-ion battery pack using MATLAB/Simulink.

## Project Description
- Cell model: A123 ANR26650M1
- Battery pack modeled in series-parallel configuration
- Simulation of voltage, temperature, current, and SOC
